## 视觉设计 (UI Design)

### 整体风格

| 属性 | 说明 |
|------|------|
| **Graphite Gray** | 主色调，石墨灰背景与中性色层级 |
| **Primary Blue** | 强调色，用于主按钮、链接、选中态 |
| **Minimal** | 极简布局，减少视觉噪音，留白充足 |
| **Card Style** | 内容以卡片容器承载，模块边界清晰 |
| **Rounded** | 统一圆角，柔和现代感 |
| **Shadow Small** | 轻微阴影，卡片与浮层层次感 |
| **Responsive** | 响应式布局，适配多种屏幕 |
| **Desktop First** | 优先桌面端设计，再向下适配平板/手机 |
| **Dark Mode** | 支持深色模式，跟随系统或手动切换 |

### 圆角与阴影

| Token | 值 | 用途 |
|-------|-----|------|
| `--radius` | `0.5rem` (8px) | 默认圆角（按钮、输入框） |
| `--radius-lg` | `0.75rem` (12px) | 卡片圆角 |
| `--shadow-sm` | `0 1px 2px rgba(0,0,0,0.05)` | 卡片阴影（Shadow Small） |
| `--shadow-md` | `0 4px 6px rgba(0,0,0,0.07)` | 下拉/弹窗 |

### 排版

| 级别 | 字号 | 字重 |
|------|------|------|
| 页面标题 | 24px / `text-2xl` | 600 |
| 卡片标题 | 18px / `text-lg` | 600 |
| 正文 | 14px / `text-sm` | 400 |
| 辅助文字 | 12px / `text-xs` | 400 |

字体栈：系统无衬线字体（`Inter`, `system-ui`, `-apple-system`, `sans-serif`）

## 布局结构

```
┌──────────────────────────────────────────────────────┐
│  Top Bar：Logo · 全局搜索 · 语言 · 主题 · 用户      │
├────────┬─────────────────────────────────────────────┤
│        │                                             │
│ Side   │  Main Content                               │
│ bar    │  ┌─────────────────────────────────────┐   │
│ Nav    │  │  Card                                │   │
│        │  │  Page Title + Actions                │   │
│        │  │  ─────────────────────────────────  │   │
│        │  │  Content (Table / Form / Dashboard)  │   │
│        │  └─────────────────────────────────────┘   │
│        │                                             │
└────────┴─────────────────────────────────────────────┘
```

### Sidebar Navigation

- 左侧固定侧边栏，可折叠（Desktop 默认展开）
- 菜单项按 RBAC 动态渲染，图标 + 文字
- 当前路由高亮（Primary Blue 左边框或背景）
- 分组：项目、采购、仓库、财务、文档、系统

### Top Search

- 顶栏居中或偏左放置全局搜索框
- 支持跨模块全文搜索（项目、材料、合同、文档等）
- 输入时实时展示下拉建议，回车跳转搜索结果页
- 搜索结果受 RBAC 约束

### Command Palette

- 快捷键 `Ctrl+K` / `Cmd+K` 唤起
- 基于 Shadcn `Command` 组件（cmdk）
- 功能：
  - 快速跳转页面（导航）
  - 全局搜索业务数据
  - 快捷操作（新建材料、新建付款单等）
  - 切换语言 / 深色模式
- 模糊匹配 + 键盘上下选择 + 回车确认

### Dark Mode

- 切换方式：顶栏按钮 / Command Palette
- 实现：Tailwind `dark` class 策略（`next-themes`）
- 偏好存储于 localStorage，下次访问保持
- 所有 Shadcn 组件使用 CSS 变量，自动适配

### Responsive（Desktop First）

| 断点 | 宽度 | 布局 |
|------|------|------|
| `lg`+ | ≥1024px | 侧边栏展开 + 顶栏搜索 + 多列卡片 |
| `md` | 768–1023px | 侧边栏可折叠，内容区单列 |
| `sm` | <768px | 侧边栏抽屉式，顶栏精简 |

## 组件规范

### Card

- 所有页面主内容区使用 Card 容器
- 结构：`CardHeader`（标题 + 操作按钮）+ `CardContent`
- 圆角 `--radius-lg`，阴影 `--shadow-sm`
- 列表页、表单页、仪表盘统一使用

### 按钮

| 类型 | 样式 |
|------|------|
| 主操作 | Primary Blue 填充 |
| 次要操作 | 描边或 Ghost |
| 危险操作 | 红色变体（删除、驳回） |

### 表格

- Shadcn `DataTable` 封装
- 表头固定，支持排序、筛选
- 行悬停浅灰/浅蓝背景
- 底部分页器

### 表单

- 标签左对齐或顶部对齐
- 必填字段标 `*`
- 校验错误红色提示于字段下方
- 提交/取消按钮居右

## 多语言

系统界面与业务文案支持三种语言：

- 中文（默认）
- 法语
- 英语

要求：

- 用户可在个人设置中切换界面语言
- 所有用户可见文案通过 i18n 管理，禁止硬编码
- 日期、数字、货币格式随语言/地区本地化

## 全文搜索

所有列表页与主要业务页面须提供全局/页面级全文搜索：

- 支持关键词模糊匹配（项目名、编号、供应商、文档标题等）
- 搜索结果高亮匹配项
- 搜索范围受 RBAC 约束，仅返回用户有权查看的数据
- 顶栏搜索与 Command Palette 共享搜索 API

## 数据导出

所有数据列表页须提供 **导出 Excel** 功能：

- 导出内容与当前筛选条件一致
- 导出列包含多币种金额的原币与折算信息（如适用）
- 导出操作记入操作日志
- 导出按钮置于 Card 标题栏右侧

## 通用交互

- 列表页：分页、排序、筛选、列配置
- 表单页：必填校验、草稿保存（如适用）
- 详情页：操作历史、关联数据 Tab
- 金额展示：同时显示币种符号与金额，支持本位币折算
- 加载态：Skeleton 占位
- 空状态：图标 + 引导文案 + 操作按钮

## 角色与界面

不同角色登录后展示对应功能菜单，菜单项与按钮可见性由 RBAC 动态控制。

## 前端实现要点

| 项目 | 方案 |
|------|------|
| 主题变量 | `globals.css` CSS 变量 + Tailwind 配置 |
| 深色模式 | `next-themes` + `dark` class |
| 侧边栏 | Shadcn `Sidebar` 或自定义 Layout 组件 |
| 命令面板 | Shadcn `Command` + 全局 `Ctrl+K` 监听 |
| 顶栏搜索 | 自定义 `SearchInput` + debounce API |
| 卡片 | Shadcn `Card` |
| 响应式 | Tailwind 断点，`lg:` 优先桌面布局 |
