# 审批与工作流模块 (workflow)

> 跨模块审批流：采购、付款、报销、合同、图纸等。

## 功能

| 功能 | 说明 |
|------|------|
| 审批提交 | 业务单据提交审批 |
| 审批处理 | 通过 / 驳回 |
| 待办列表 | 当前用户待审批事项 |
| 已办列表 | 当前用户已处理事项 |
| 我发起的 | 发起人查看进度 |
| 审批记录 | 完整审批链历史 |
| 流程配置 | 审批模板管理（管理员） |
| 通知 | 待办通知（站内信，BullMQ） |
| 全文搜索 | 单号、发起人、摘要 |
| Excel 导出 | 审批记录导出 |

## 字段

### 审批实例 (approval_instances)

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `code` | string | 是 | 审批单号 |
| `type` | enum | 是 | 业务类型（见下表） |
| `businessId` | uuid | 是 | 业务单据 ID |
| `projectId` | uuid | 否 | 关联项目 |
| `initiatorId` | uuid | 是 | 发起人 |
| `status` | enum | — | `pending` \| `approved` \| `rejected` \| `cancelled` |
| `currentNode` | int | — | 当前审批节点 |
| `createdAt` | datetime | — | 创建时间 |

### 业务类型 (type)

| 值 | 说明 | 审批链 |
|----|------|--------|
| `purchase_request` | 采购申请 | 项目主管 |
| `payment` | 付款 | 财务 → 老板（超额度） |
| `reimbursement` | 报销 | 项目主管 → 财务 |
| `contract` | 合同签订 | 项目主管 → 财务 → 老板 |
| `drawing` | 图纸发布 | 工程师 → 项目主管 |

### 审批记录 (approval_records)

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `instanceId` | uuid | 是 | 审批实例 |
| `node` | int | 是 | 节点序号 |
| `approverId` | uuid | 是 | 审批人 |
| `action` | enum | 是 | `approve` \| `reject` |
| `comment` | text | 否 | 审批意见 |
| `actedAt` | datetime | — | 操作时间 |

### 流程模板 (approval_templates)

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `type` | enum | 是 | 业务类型 |
| `nodes` | json | 是 | 节点配置（角色、条件） |
| `name` | string | 是 | 模板名称 |

## 业务规则

- **付款必须审批** — 无审批通过不得执行付款
- **采购申请必须经项目主管审批**
- **报销申请必须经项目主管和财务审批**
- 审批人按角色 + 项目动态解析
- 驳回后发起人可修改重新提交
- 每次审批动作记入 audit_log

## API

基础路径：`/api/v1/workflow`

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/approvals` | 审批列表（筛选 type/status） |
| GET | `/approvals/:id` | 审批详情 |
| POST | `/approvals` | 创建审批（通常由业务模块触发） |
| POST | `/approvals/:id/approve` | 审批通过 |
| POST | `/approvals/:id/reject` | 审批驳回 |
| POST | `/approvals/:id/cancel` | 撤回 |
| GET | `/approvals/todo` | 我的待办 |
| GET | `/approvals/done` | 我的已办 |
| GET | `/approvals/initiated` | 我发起的 |
| GET | `/approvals/export` | Excel 导出 |

### 流程模板（管理员）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/templates` | 模板列表 |
| POST | `/templates` | 创建模板 |
| PUT | `/templates/:id` | 编辑模板 |

## 权限

| 操作 | 管理员 | 项目主管 | 财务 | 老板 | 发起人 |
|------|--------|----------|------|------|--------|
| 审批处理 | ✓ | 对应节点 | 对应节点 | 对应节点 | — |
| 查看待办/已办 | ✓ | ✓ | ✓ | ✓ | — |
| 查看我发起的 | ✓ | ✓ | ✓ | ✓ | ✓ |
| 模板管理 | ✓ | — | — | — | — |
| 撤回 | ✓ | — | — | — | 本人（pending 时） |
| Excel 导出 | ✓ | ✓ | ✓ | ✓ | ✓（本人） |

## 操作日志

- 审批提交、通过、驳回、撤回
- 模板变更
- 导出
