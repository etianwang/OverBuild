# 业务域

## 背景

OverBuild 面向国内公司的海外驻外工程项目，提供统一的内部管理平台，覆盖项目全生命周期的核心业务。

## 业务模块

| 模块 | 职责 |
|------|------|
| 登录权限 | 认证、RBAC、用户与角色管理 |
| 项目管理 | 项目立项、进度、人员、成本归集 |
| 材料 | 材料主数据、分类、价格、库存预警 |
| 材料采购 | 采购申请、询价、订单、到货跟踪 |
| 仓库库存 | 入库、出库、盘点 |
| 合同 | 合同签订、变更、履约跟踪 |
| 财务 | 收支、报销、付款、多币种核算 |
| 文档 | 项目文档归档、版本、权限 |
| 图纸 | 工程图纸上传、审阅、版本控制 |
| 翻译 | 多语言文档与界面内容翻译 |
| 审批 | 跨模块审批流 |
| Dashboard | 首页数据概览 |
| 通知 | 待办、业务提醒 |
| 系统设置 | 全局配置、个人偏好 |
| 日志 | 操作审计 |
| 成本分析 | 项目成本汇总（归集于项目/财务/Dashboard） |

## 用户角色

| 角色 | 典型职责 |
|------|----------|
| 管理员 | 系统配置、用户与角色管理、全局审计 |
| 老板 | 全局数据查看、审批、成本分析 |
| 项目经理（项目主管） | 项目创建与推进、资源协调、成本监控、采购/报销审批 |
| 采购 | 采购申请处理、供应商对接、订单管理 |
| 仓库管理员 | 出入库、盘点、库存维护 |
| 财务 | 账务处理、付款审批、多币种结算 |
| 工程师 | 技术文档、图纸提交与查阅 |
| 翻译 | 文档与术语翻译、多语言维护 |

## 跨模块规则

- 所有角色通过 RBAC 控制可见范围与操作权限
- 项目为多数业务数据的归属维度（项目 ID 贯穿各模块）
- 金额字段统一支持多币种，需记录币种与汇率
- 关键业务动作（创建、修改、删除、审批、导出）必须写操作日志

## 业务规则 (Business Rules)

### 项目

- 一个项目可以包含多个合同
- 一个项目可以对应多个仓库
- 一个项目可以对应多个采购订单
- 一个项目可以拥有多个施工区域

### 材料

- 库存不得小于零
- 材料只能属于一个分类
- 每次出库必须关联项目
- 每次采购必须关联供应商
- 材料价格保留历史记录

### 财务

- 付款必须审批
- 回款必须关联合同
- 项目利润实时计算
- 汇率每日更新
- 预算不能删除
- 采购申请必须经项目主管审批
- 报销申请必须经项目主管和财务审批

### 文档

- 所有文档保存版本
- 支持 PDF、DWG、图片、Office 格式
- 文档可预览

### 翻译

- 所有字段支持中法双语
- 支持自动翻译
- 保留人工翻译版本

### 审批

- 付款必须审批
- 采购申请必须经项目主管审批
- 报销必须经项目主管和财务审批
- 图纸发布须经审阅批准
